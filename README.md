# 🧾 Bio Data Viewer

A simple and user-friendly Flutter application that allows users to enter personal details and view them in a beautifully formatted biodata layout.

---

## 📱 Features
- Enter personal details easily using text fields.  
- Display biodata in a clean and attractive format.  
- Organized and beginner-friendly Flutter code.  
- Modular structure using a separate `bio_data.dart` file.

---

## 🚀 Getting Started

### 1️⃣ Prerequisites
Make sure you have:
- Flutter SDK installed → [Flutter Installation Guide](https://flutter.dev/docs/get-started/install)
- Android SDK (via Android Studio or Command Line Tools)
- A connected emulator or physical device

---

### 2️⃣ Installation Steps

Clone this repository:
```bash
git clone https://github.com/<your-username>/bio_data_viewer.git
```

Navigate into the project folder:
```
cd bio_data_viewer
```

Get all dependencies:
```
flutter pub get
```

Run the app:
```
flutter run
```
🧩 Project Structure
```
bio_data_viewer/
│
├── lib/
│   ├── main.dart          # Entry point
│   ├── bio_data.dart      # Contains the biodata display UI
│
├── android/               # Android-specific files
├── ios/                   # iOS-specific files
├── pubspec.yaml           # Project dependencies
└── README.md              # You are here 😄
```
🧠 How It Works

main.dart → Imports and sets up BioDataPage as the home screen.

bio_data.dart → Displays formatted biodata details entered by the user.
